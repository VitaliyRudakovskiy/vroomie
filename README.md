## Vroomie — цифровой гараж для учёта обслуживания автомобиля

**Vroomie** — это web‑приложение (SPA) с mobile‑first подходом, которое помогает автовладельцам вести историю обслуживания своих машин.

Приложение позволяет добавлять автомобили, фиксировать выполненные ремонты, планировать будущие работы, хранить фото, чеки и ссылки на запчасти или видео‑инструкции.

### Основные возможности

- Регистрация и авторизация через Firebase.
- Добавление автомобилей с фото и базовой информацией.
- Ведение истории ремонтов: пробег, дата, заметки, фотографии.
- Планирование будущих работ: описание, ссылки на запчасти и видео.
- Перевод планов в выполненные ремонты.
- Хранение изображений в Cloudinary.
  Поддержка системы друзей.

### Технологии

- Angular 20
- NgRx
- Firebase Auth
- Firebase Firestore
- Cloudinary
- RxJS
- Biome
